# Simple Project
super awesome swag

## howto

1. `npm install`
2. `gulp`

## what it does

- it creates a `dist` folder for all your css and such
- it turns stylus into css
- it concats all your css
- it compresses all your css
- it has a webserver built in
- you can clean your dist folder with `gulp clean`
